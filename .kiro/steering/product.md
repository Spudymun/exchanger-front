# Product Overview

## ExchangeGO - Enterprise Crypto Exchange Platform

A modern cryptocurrency exchange platform built as a monorepo with multiple applications and shared packages.

### Core Applications

- **Web App** (`apps/web`) - Main customer-facing exchange platform (localhost:3000)
- **Admin Panel** (`apps/admin-panel`) - Administrative dashboard for managing the exchange (localhost:3002)
- **Documentation** (`apps/docs`) - Technical documentation and guides (localhost:3001)

### Key Features

- **Multi-currency trading** - Support for Bitcoin, Ethereum, USDT, Litecoin and other cryptocurrencies
- **Real-time trading** - Live order books, price feeds, and trading execution
- **User management** - Account creation, KYC verification, role-based access
- **Order management** - Buy/sell orders, order history, portfolio tracking
- **Admin tools** - User management, transaction monitoring, system configuration
- **Internationalization** - Multi-language support (English, Russian)

### Business Domain

The platform handles cryptocurrency exchange operations including:

- User registration and authentication
- Trading pair management (BTC/USD, ETH/BTC, etc.)
- Order processing and matching
- Portfolio and balance management
- Transaction history and reporting
- Administrative oversight and compliance

### Target Users

- **Traders** - Individual and institutional cryptocurrency traders
- **Administrators** - Exchange operators managing users and transactions
- **Developers** - Technical team maintaining and extending the platform
