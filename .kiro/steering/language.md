# Языковые предпочтения и локализация

## Предпочтительный язык общения

**Русский язык** - основной язык для общения с разработчиками и документации.

### Правила общения

- **Отвечай на русском языке** когда пользователь пишет на русском
- **Используй техническую терминологию** на английском только когда это необходимо (названия технологий, команды, код)
- **Объясняй концепции** на русском языке для лучшего понимания
- **Комментарии в коде** могут быть на русском языке для внутренней разработки

### Поддерживаемые локали

Проект поддерживает интернационализацию через next-intl:

- **Русский (ru)** - основной язык для российских пользователей
- **Английский (en)** - международный язык по умолчанию

### Структура переводов

```
apps/web/messages/
├── ru.json    # Русские переводы
└── en.json    # Английские переводы
```

### Примеры использования

#### В коде:

```typescript
// Комментарии на русском для внутренней команды
const ORDER_STATUSES = {
  PENDING: 'pending', // Ожидает подтверждения
  CONFIRMED: 'confirmed', // Подтвержден
  SHIPPED: 'shipped', // Отправлен
} as const;
```

#### В документации:

- Техническая документация может быть на русском языке
- README файлы содержат описания на русском
- Комментарии к коммитам могут быть на русском

### Исключения

Следующие элементы остаются на английском:

- Названия переменных, функций, классов
- Git commit messages (желательно)
- Технические команды и конфигурации
- Названия пакетов и зависимостей

### Контекст проекта

Поскольку это российская криптовалютная биржа ExchangeGO, русский язык является естественным выбором для:

- Общения с командой разработки
- Внутренней документации
- Пользовательского интерфейса для российских клиентов
- Бизнес-логики и требований
