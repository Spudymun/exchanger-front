### Путь: packages/utils/src/validation/constants.ts

**Краткое назначение (1 предложение)**
Константы валидации и конфигурация для интеграции с next-intl системой переводов.

**Подробное описание (3–6 предложений)**
Файл содержит централизованные ключи переводов для валидации, устраняя дублирование строк в системе. Определяет интерфейс NextIntlValidationConfig для типобезопасной интеграции с next-intl. Включает ключи для всех типов валидации (email, пароли, CAPTCHA, суммы) с семантическими именами. Обеспечивает единый источник истины для ключей переводов валидации. Поддерживает локализацию через функцию перевода с параметрами.

**Экспортируемые сущности / API (все на 100%)**

- `VALIDATION_KEYS` - объект с ключами переводов (INVALID, REQUIRED, EMAIL_INVALID, PASSWORD_MIN_LENGTH и др.)
- `NextIntlValidationConfig` - интерфейс конфигурации (t: функция перевода, locale?: строка)

**Оценка сложности (low/medium/high)**
**LOW** - константы и интерфейсы без логики
