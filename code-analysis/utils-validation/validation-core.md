### Путь: packages/utils/src/validation/core.ts

**Краткое назначение (1 предложение)**
Основные функции валидации с интеграцией next-intl для создания локализованных Zod error maps.

**Подробное описание (3–6 предложений)**
Файл содержит ключевые функции для интеграции Zod валидации с next-intl системой переводов. Включает createNextIntlZodErrorMap для создания локализованных карт ошибок и validateFormWithNextIntl для валидации форм с переводами. Использует специализированные обработчики для различных типов полей (CAPTCHA, email, пароли, суммы). Обеспечивает единый источник истины для переводов через next-intl. Следует принципу централизованной обработки ошибок валидации.

**Экспортируемые сущности / API (все на 100%)**

- `createNextIntlZodErrorMap(config)` - создание локализованной карты ошибок Zod
- `validateFormWithNextIntl<T>(schema, data, config)` - валидация форм с переводами
- Вспомогательная функция: `handleValidationIssue` (внутренняя)

**Оценка сложности (low/medium/high)**
**MEDIUM** - интеграция Zod с next-intl и обработка различных типов ошибок
